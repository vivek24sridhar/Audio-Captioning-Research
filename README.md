# Audio-Captioning-Research
Hierarchical audio captioning system that converts complex acoustic scenes into text using YAMNet for sound classification, Whisper for speech transcription, and GraphAC for non-speech audio, enabling clear descriptions of multi-sound environments.
Here the input audio can also be given as a file or can be recorded lively. Usage of YAMNet enhances to give input audio in any language and it translates to english and then print as output.
